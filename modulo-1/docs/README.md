# Módulo 1 — Fundamentos de TypeScript

Este módulo introduce los conceptos esenciales para trabajar con TypeScript en un proyecto real
El objetivo es aprender a:

- Crear un proyecto con estructura profesional
- Configurar TypeScript y Node.js con ESModules
- Escribir funciones tipadas
- Manejar casos especiales (arrays vacíos)
- Compilar TypeScript a JavaScript
- Ejecutar el proyecto con `tsx`
- Documentar el código de forma clara
